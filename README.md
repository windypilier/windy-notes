# Windy Notes

Application d'interprétation trilingue avec versets bibliques quotidiens.

## Fonctionnalités

- 🌍 Support trilingue (Français A, Espagnol B, Anglais C)
- 📖 Versets bibliques quotidiens
- 🎙️ Outils d'interprétation
- 📝 Prise de notes
- 💾 Mode hors ligne
- 🌙 Mode sombre

## Langues supportées

- 🇫🇷 Français (Langue A)
- 🇪🇸 Espagnol (Langue B)
- 🇬🇧 Anglais (Langue C)

## Technologies utilisées

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
